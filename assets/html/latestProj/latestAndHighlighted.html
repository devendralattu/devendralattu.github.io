Shopping Planner using TSP

Developed a web-application to implement a flavor of TSP to traverse different shops.

New

Implemented the Nearest Neighbor algorithm to find a shortest route to traverse shops from user's home location. Used MongoDB geospatial queries and aggregation pipeline to filter the input. Created RESTful API in NodeJS and used Google Maps API to display the resultant route.
Dataset: Simple Geo Public Spaces, archive.org.
